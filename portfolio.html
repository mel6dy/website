1.1.9 Grow Your Own Garden


This program essentially lets the user grow their own garden. There are four flowers in total, and the user is able to customize each flower's color to their liking. The progam was built using many methods that we learned in this unit such as for loops, while loops, and lists. Once all four flowers are grown, the user is met with a happy rainbow :)


1.2.5 Space Invaders


This is a game called Space Invaders where your galaxy is under attack by aliens! Except theses aliens are different... they throw math questions at you and you must answer them correctly to defeat them! You have 20 seconds to kill as many aliens as possible, and if your score is high enough, your name will make the leaderboard.


Scratch Project Cooking Game


This game is a cooking game where the user gets to choose between a steak and a burger. We made this scratch program by utilizing the costumes feature on Scratch which changed the costume of the meat when it "cooked." We also added a timer on the burger, and included more than one background for the kitchen and welcome screen. The broadcast funciton was really useful in the creation process as that was how we communicated to the program which food the player picked and whenever they finished each step.
